What is Accme?

Accme is a model that uses an algorithm and computer vision to detect if an accessible vehicle is present in a video stream by looking for the International Symbol of Access (ISA) in places such as license plates and windows.

Model Performance Overview

Speed

Accme is optimized for real-time processing. It can process frames at a rate of 15fps.

Memory

Accme utilizes a dataset with 55 pictures, ensuring low memory usage.

Accuracy

Accme has an impressive detection accuracy rate of 66% and the confidence rate of over 45%. Our model ensures that accessible vehicles are reliably identified.
